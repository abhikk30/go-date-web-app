# go-date-web-app

Generated by the lw-idp Forge. A standalone TypeScript service (Fastify + ConnectRPC).

- Health: `GET /healthz`, `GET /readyz`
- Metrics: `GET /metrics`
- Listens on port `8080`.

> The ConnectRPC handler (and its `proto/` service) is the generated domain
> layer — it is wired into the entrypoint when the service is authored, not in
> the bare skeleton.

CI (Jenkins + kaniko) builds the image and bumps `chart/values.yaml` `image.tag`;
Argo CD syncs `chart/` to the `go-date-web-app` namespace.
